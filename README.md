# WanderWise ‚Äì Property Rental & Review Platform

This is a full-stack web application for renting villas and Review Platform.
Built using **Node.js, Express, MongoDB, and EJS templates**.

## Features
- Add, edit, delete villa and resort listings  
- Upload images using Cloudinary  
- User login and signup  
- Reviews and ratings system  
- Map integration for locations  
- Fully responsive pages  

## Technologies Used
- Node.js  
- Express.js  
- MongoDB (Atlas)  
- Cloudinary  
- EJS  
- Bootstrap / CSS
